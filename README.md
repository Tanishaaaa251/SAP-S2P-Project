## SAP S2P Project Title - Simulation of Source-to-Pay (S2P) Process using Structured Workflow and Data Modeling.

This project illustrates a simulation of the Source-to-Pay (S2P) process.

### 📌 Features:
- Selection of vendors on the basis of price, delivery and rating.
- RFQ comparison
- Purchase Order creation
- Goods Receipt tracking
- Payment processing

### 📂 Files Included:
- S2P_Project.xlsx - Data simulation
- M.Tanisha_SAP_S2P_Consultant_ProjectDocumentation.pdf - Complete documentation
- SAP_S2P_Project_Tanisha.zip - Full project files.

### 📊 Description:
The project emulates an end-to-end procurement cycle with structured data and workflow design, on the concepts of SAP S/4HANA.
